# Embeddings

## Overview

Embeddings are numerical representations of text, code, images, or other data in a high-dimensional vector space.

They allow AI systems to understand semantic relationships between different pieces of information.

---

## Why Embeddings are Important

Traditional keyword search only matches exact words.

Embeddings allow semantic search:

Example:

Query:
"How to create an AI assistant?"

Can retrieve:

"Building a conversational agent using LLMs"

because the meaning is similar.
