A curated collection of research notes, literature reviews, implementation insights, experiments, and learning resources focused on **Artificial Intelligence**, **Large Language Models (LLMs)**, **Retrieval-Augmented Generation (RAG)**, **AI Agents**, and **Software Engineering**.

This repository serves as my personal research journal, documenting concepts, experiments, paper reviews, and implementation ideas while exploring AI-driven software engineering.
